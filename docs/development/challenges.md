## 인터페이스 설계 과제
### 1. 통합 상호작용 시스템
```cpp
// 다양한 상호작용을 하나의 인터페이스로 통합
UENUM(BlueprintType)
enum class InteractType : uint8 {
    NONE, STATIC, ITEM, NPC
};
```
- **도전 과제**: 서로 다른 상호작용 타입을 단일 키로 처리하면서도 각각의 고유한 동작을 유지해야 했음
- **해결 방안**: 인터페이스 기반 설계를 통해 확장 가능한 구조 구현
